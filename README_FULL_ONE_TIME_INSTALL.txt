FULL ONE-TIME INSTALL VERSION — B1 Goethe Vocabulary Trainer

This is the corrected full package.
It contains the full vocabulary app plus PWA/offline cache files.

Install:
1. Unzip this ZIP.
2. Upload the UNZIPPED folder contents to Netlify Drop / GitHub Pages / Cloudflare Pages.
3. Open the generated link in Chrome.
4. Wait 5–10 seconds so the service worker caches the app.
5. Add to Home Screen.
6. Open the Home Screen app.
7. Test airplane mode. If it opens, the offline install worked.

Important:
- iPhone cannot install a ZIP directly as a native app.
- You need the hosted link once for the first install.
- After the offline cache is installed, the Home Screen version should continue opening even if the host later has issues.
- Do not clear Chrome/Safari website data, otherwise progress/cache can be lost.
